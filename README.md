# React Epics, Features & User Stories Viewer

This is a React 18 project set up with Vite to visualize Epics, Features, and User Stories from JSON data.

## Setup & Run

1. Install dependencies:

npm install

2. Start development server:

npm run dev

3. Visit http://localhost:5173 to see the app.

## Files

- `src/main.jsx` — React DOM entry point.
- `src/App.jsx` — Main app component.
- `src/components/Epic.jsx` — Epic component.
- `src/components/Feature.jsx` — Feature component.
- `src/components/UserStory.jsx` — User Story component.
- `src/App.css` — Stylesheet.

Enjoy!
