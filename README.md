# React Epics, Features & User Stories Viewer

This is a React 18 application built with Vite that visualizes project Epics, Features, and User Stories from JSON data.

## Setup & Run

1. Install dependencies:

npm install

2. Start the development server:

npm run dev

3. Open the URL in your browser (usually http://localhost:5173)

## File Structure

- `src/main.jsx`: Entry point
- `src/App.jsx`: Main app component
- `src/components/Epic.jsx`: Renders an Epic and its Features
- `src/components/Feature.jsx`: Renders a Feature and its User Stories
- `src/components/UserStory.jsx`: Renders User Story details
- `src/App.css`: Styles

Enjoy exploring your project data!
