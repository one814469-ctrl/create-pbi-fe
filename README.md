This project implements a fully traceable, dynamically-mapped React application using the SLPS PRD and PBIs.

## Features

- Dynamic epic routing and navigation
- User stories rendered as interactive feature blocks
- Tasks implemented with functional UI aligned to PRD
- Mock integrations (API, upload, notifications)
- Demo-only compliance with business requirements

## Getting Started

1. Install dependencies:
   
   npm install
   

2. Run the application:
   
   npm run dev
   

3. Visit [http://localhost:5173](http://localhost:5173) or as instructed in the terminal.

## Structure

- Epics, user stories, and tasks sourced from `src/data/epics.json`
- All features are traceable to PRD as documented.

---

For any missing PBIs or PRD clarifications, update `src/data/epics.json` and re-run.
