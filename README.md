# Smart Loan Processing System

## Setup Instructions

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
4.  Open your browser and navigate to `http://localhost:5173`.

## Project Structure

-   `src/`: Contains all the source code.
    -   `App.jsx`: The main application component.
    -   `components/`: Contains reusable React components.
        -   `EpicPage.jsx`: Displays details for a specific epic.
        -   `UserStoryBlock.jsx`: Displays details for a specific user story.
        -   `TaskComponent.jsx`: Displays a task with interactive elements.
    -   `data/`: Contains JSON data for epics and user stories.
        -   `epics.json`: Contains the epic, user story, and task data.

## Dependencies

-   React
-   React Router DOM
-   Tailwind CSS
-   Axios
